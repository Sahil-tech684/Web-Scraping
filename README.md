🕷️ Web Scraping Collection

A collection of Python web scraping and browser automation projects built using Selenium, Scrapy, BeautifulSoup, Requests, and other scraping tools.

---

📖 Overview

This repository contains my personal collection of web scraping projects developed while learning and working with different websites, technologies, and anti-bot challenges.

The goal of this repository is to explore various scraping techniques, automate data collection, and build reusable scraping solutions for static and JavaScript-heavy websites.

Each project focuses on solving a different scraping problem—from simple HTML extraction to handling dynamic pages, pagination, lazy loading, browser automation, and structured data export.
